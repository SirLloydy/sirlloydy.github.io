< table tags >

Tag Name Description
<table> Table The wrapper element for all HTML tables.
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<tr> Table Row The table row container.
<td> Table Data The table data displayed.
<tfoot> Table Foot The set of rows defining the footer in a table

< table attributes >

Attribute Name Description
colspan Column Span Defines how many columns a td elemtn should span.
rowspan Row Span Defines how many rows a td element should span.

< semantic layout elements >

Element Name Description
<html> Root The top-level element thats wraps the entire html document.
<head> Metadata Container Contains non-visible data like page title, character set and css links.
<body> Document Body Contains all the visible content of the webpage.
<header> Header Used for introductory content, logos, or navigation at the top of a page/section.
<nav> Navigation Specifically designates a block of navigation links.
<main> Main Content Represents the unique, primary content of the document (only one per page).
<section> Section Defines a generic standalone section of a document, usually with a heading.
<article> Article Defines independent, self-contained content (e.g., a blog post or news story)
<aside> Aside Content indirectly related to the main text, often used for sidebars or pull-quotes.
<footer> Footer Contains "bottom-of-page" info like copyright, contact details, or site maps.
<div> Division A generic container used for styling or layout when no semantic tag fits.
